- Itens estudados:
    - Relacionamento @ManyToOne
    - Relacionamento @OneToMany
    - Relacionamento @ManyToMany
    - EAGER e LAZY(com JOIN FETCH)

    - @NamedQuery
    - Criteria API
    - Gerar relatório(com SELECT NEW)
    - Consultas com JPQL
